# CSS Animations Collection

A comprehensive collection of CSS animations, transitions, and hover effects organized by type. This project showcases various button animations, image effects, navbar hover effects, and card hover effects created with pure CSS.

- Cross background button
- Letter spacing button
- Glowing button
- Two layer swipe button
- Text scale button

- Rotating layers button
- Layers swapping button
- Circles background button
- 3D rotate button
- Splited button
- Grid button
- Rotating letter button
- Cuts button

- Pulsing button
- Diagonal swipe button
- Glowing lines button
- Glass button

### Image Hover Effects

### Navbar Hover Effects

### Card Hover Effects

## Getting Started

1. Clone the repository:
```bash
git https://github.com/vasylpryimakdev/css-animations-collection.git
cd css-animations-collection
```

2. Open `index.html` in your browser to see all demos

3. Navigate to specific effect folders to see individual demos
